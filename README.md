# E-bookstore Management System

This project is a simple console-based bookstore management system implemented in Python using SQLite for data storage. It allows users to perform basic operations like adding, updating, searching, deleting books, and viewing the entire book inventory.

## Features

1. **Add Book:** Users can input book details like title, author, and quantity to add new books to the database.

2. **Update Book:** Users can update existing books by providing the title and author of the book.

3. **Search Book:** Users can search for a specific book by providing the title and author.

4. **Delete Book:** Users can delete a book from the database using the title and author.

5. **View All Books:** Users can view the details of all books stored in the database.

## Getting Started

### Prerequisites

- Python 3.x
- SQLite3

### Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/bookstore-management.git
cd bookstore-management
```

2. Run the script:

```bash
python bookstore.py
```

3. Follow the on-screen instructions to interact with the bookstore management system.

## Database

The application uses SQLite as a lightweight database. The database file is named `ebookstore.db`.

## Sample Data

The project comes with some initial sample data to demonstrate functionality. You can customize this data in the script.
